Detailed Description

Generates Price(S) and Delta(S) graphs for lookback options.

Sweeps the spot price over a user-defined range, evaluates the Monte Carlo pricer (and optionally the analytic solution) at each point, writes the results to a CSV file, and (when gnuplot is available) produces PNG plots.
